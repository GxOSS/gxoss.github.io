GxOSS
=============

Free and Open-source Xbox 360 Utilities

## Genexis

Cross-platform Xbox 360 NAND Building/Flashing and Timing Flashing; Central wrapper for all GxOSS tools + more

- [Source]()


## gxBuild

Rust Xbox 360 NAND Builder and Patcher.

## ESPFlasher

ESP32 Wireless NAND Flashing via TCP

- [Source]()


## GxDebug

ESP32 Wireless Kernel/SMC UART and POST reading

Client for ESPFlasher, PicoFlasher and GxDebug; Wireless and USB NAND/Timing Flasher and Debugging

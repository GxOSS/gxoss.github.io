GxOSS
=============

Free and Open-source Xbox 360 Utilities

## Genexis

Cross-platform Xbox 360 NAND Building/Flashing and Timing Flashing; Central wrapper for all GxOSS tools + more

- [Source](https://github.com/GxOSS/Genexis)


## gxBuild

Rust Xbox 360 NAND Builder and Patcher.

- [Source](https://github.com/GxOSS/gxBuild) - [Support Files](https://github.com/GxOSS/gxBuild-Support-Files) - [Patch Sources](https://github.com/GxOSS/gxPatch-sources) - [Patch Devtools](https://github.com/GxOSS/gxPatch-devtools)


## ESPFlasher

ESP32 Wireless NAND Flashing via TCP

- [Source](https://github.com/GxOSS/espflasher)


## GxDebug

ESP32 Wireless Kernel/SMC UART and POST reading

- [Source](https://github.com/GxOSS/GxDebug)


## FlasherClient

Client for ESPFlasher, PicoFlasher and GxDebug; Wireless and USB NAND/Timing Flasher and Debugging

- [Source](https://github.com/GxOSS/FlasherClient)
